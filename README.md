# Pesquisa de satisfação

Projeto prático rápido feito para coletar dados sobre a satisfação dos hospedes de um hotel.

## Estrutura do projeto

Nesse exemplo foram colocadas 8 questões de avaliação com [input]radio para que seja obrigatório a seleção de apenas uma alternativa. Foram definido também apenas 4 opções, com o intuito de não ter "meio termo" e a metrica ser mais efetiva.
  
Também foram colocados 2 [textarea] para receber as opiniões com as palavras do cliente e ter um resultado mais organico.   

Por ultimo foi adicionado 3 campos para identificação do usuário.

Tudo isso é mandado para o banco de dados (MYSQL) por meio de um form, com action em um arquivo chamado "envio.php".

## Objetivo

Coletar dados e aplicar em situação real.

## Tecnologias utilizadas

-HTML  
-CSS  
-Bootstrap  
-PHP  
-MySQL  

## Adicional

Esse projeto possui um painel administrativo, onde reunindo os dados retirados nesse projeto pode criar parametros de analise, configurando cenarios como por exemplo de onde o cliente veio, o que ele mais gostou, o que não gostou e setores com maior deficiencia, pra acessar o projeto basta [clicar aqui!](https://github.com/StartBootstrap/startbootstrap-sb-admin-2)
